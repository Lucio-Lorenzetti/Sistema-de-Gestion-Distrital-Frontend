# Sistema de Gestión Distrital - Frontend

Este proyecto forma parte del **Proyecto Final de Carrera** de Ingeniería en Sistemas de Información.  
El objetivo es desarrollar una plataforma web integral para el Distrito 3, Zona 13 perteneciente a la Asociación Scout de Argentina, que permita centralizar la comunicación institucional, la gestión de documentación y el trabajo colaborativo entre los grupos que la integran.
Interfaz de usuario moderna y reactiva para el Sistema de Gestión Distrital. Desarrollada para garantizar una experiencia de usuario fluida y eficiente.

## Tecnologías
* **Framework:** [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
* **Estilos:** Tailwind CSS
* **Enrutamiento:** React Router DOM
* **Cliente API:** Axios
* **Hosting:** Vercel (Producción)

## Estructura del Sitio
* **Área Pública:** Home, Galería, Listado de Cursos, Descargas y Noticias.
* **Área Privada (Dashboard):** * Gestión de programas (Mis programas / Subir programas).
    * Gestión de cursos e inscripciones.
    * Administración de usuarios y asignación de roles (según jerarquía).
